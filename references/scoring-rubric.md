# Scoring Rubric

Use this rubric after evidence collection. The goal is consistency, not theatrical precision.

Each pillar has five sub-signals. Score each sub-signal from `0` to `5`.

- `0`: absent or clearly broken
- `1`: weak and mostly ineffective
- `2`: below standard
- `3`: acceptable but unremarkable
- `4`: strong
- `5`: excellent and clearly above average

Dimension score formula:

`(sum of 5 sub-signals) * 4 = 0-100`

## SEO

Score these five sub-signals:

1. Crawl and index readiness
   Signals: status code, canonical alignment, indexability, SSR or rendered visibility, clean heading structure
2. Metadata and page semantics
   Signals: title, description, H1, semantic sectioning, descriptive anchors
3. Internal linking and discoverability
   Signals: internal links, contextual anchors, navigational depth, related content connections
4. Structured data and media hygiene
   Signals: schema coverage, image alt quality, media relevance, machine-readable hints
5. Performance and UX stability
   Signals: CWV, mobile readiness, rendering stability, low friction to consume the page

## GEO

Score these five sub-signals:

1. Entity clarity
   Signals: clear definition of brand, product, audience, problem, and topic entities
2. Topical completeness
   Signals: the page covers the key subtopics a generative system would need to answer well
3. Authority and trust signals
   Signals: proof, citations, examples, expertise markers, original insight
4. Factual grounding
   Signals: claims are specific, scoped, and not hand-wavy
5. Query coverage breadth
   Signals: the page can support multiple adjacent intents, not just the exact headline phrase

## AEO

Score these five sub-signals:

1. Answer-first clarity
   Signals: concise direct answers appear early and often
2. Snippet readiness
   Signals: extractable paragraphs, lists, tables, definitions, and summary blocks
3. Question handling
   Signals: explicit question-led subheads, FAQ quality, objection handling
4. Scanability
   Signals: clear hierarchy, short sections, low cognitive load
5. Intent match
   Signals: the page answers what the searcher likely meant, not just what the keyword literally says

## LLMO

Score these five sub-signals:

1. Chunkability
   Signals: sections stand on their own and preserve meaning when retrieved in isolation
2. Quoteability
   Signals: passages can be quoted or paraphrased cleanly without ambiguity
3. Retrieval anchors
   Signals: precise terminology, aliases, definitions, and headings that help matching
4. Semantic disambiguation
   Signals: the page removes confusion around brand names, acronyms, product scope, and audience
5. Reusability
   Signals: content can fuel summaries, comparisons, recommendations, and generated answers

## Interpretation Notes

- A visually polished page can still score poorly if it lacks extraction-friendly structure.
- A keyword-heavy page can still score poorly if it lacks authoritative or reusable substance.
- A technically clean page can still score poorly if its answers are vague or shallow.
- A content-only audit should avoid over-scoring technical dimensions.
